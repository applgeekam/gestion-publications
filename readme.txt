## Context du projet : 
Ce projet est une application bureau qui sert a enregistrer des publication de memoires ou de theses effectue par les etudiants. On peut aussi consulter la listes des publications, modifier ou voir le document relatif a une publication.


## Contenu du depot

- Dans le dossier module, il y a essentiellement les resultat de mes recherches.

- Dans publication, il y a le code source. C'est une application ecrit en C++ avec le framework Qt 5. Elle utilise en backend, une db Access, mais on peut aussi utiliser une db Mysql. Les documents importer dans l'application sont stocker dans un dossier Publications se trouver dans le dossier personnel de l'utilisateur du PC.
